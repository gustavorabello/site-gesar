--- title: Template LaTeX para tese de doutorado PPG-EM description: Template LaTeX para tese de doutorado PPG-EM created: !!timestamp '2016-08-13 5:00:00' --- {% mark img1 -%} ![s1]([[!!images/logos/latex.png]]) {%- endmark %} {% mark excerpt -%} Este é um Template LaTeX para teses de doutorado do PPG-EM/UERJ que preenche os requisitos de formatação especificados pela biblioteca do CTC/B. Este template tem como base o modelo de template do Programa Doutorado de Engenharia Elétrica da UERJ. No arquivo .zip também está o Roteiro de Teses e Dissertações da UERJ para conferência e adaptação para outros programas. - [template em LaTeX - inglês][1] - [PDF gerado - inglês][3] Note que para usuários de linha de comando em sistemas Linux/Unix, um Makefile está disponível para compilação automática dos arquivos LaTeX e BibTeX. Todos os arquivos podem ser alterados e modificados para atender a qualquer tipo de pré-requisito de tese de doutorado, tendo os autores como responsáveis por sua manutenção.
