--- title: Instrução para publicação de mini artigo no anais do Seminários do PPG-EM description: Anais do Seminário do PPG-EM created: !!timestamp '2016-01-26 5:00:00' --- {% mark img1 -%} ![s1]([[!!images/logos/seminarios.png]]) {%- endmark %} {% mark excerpt -%} Estamos organizando uma edição com todos os seminários do PPG-EM apresentados no ano atual. Se você fez uma apresentação recentemente, porém ainda não fez seu artigo de 2 páginas, gostaríamos de incentivá-lo a preparar este documento em inglês contendo APENAS 2 páginas para publicarmos nos anais dos Seminários do PPG-EM. Este trabalho será publicado na página do GESAR, no seguinte endereço: - [www.gesar.uerj.br/cursos][1] Para elaboração do documento, um template foi criado para padronizar o conteúdo dos artigos e facilitar o desenvolvimento da publicação final, contendo todos os seminários apresentado no ano. Este template pode ser obtido em: - [template em LaTeX][2] Atualmente apenas template em LaTeX está disponibilizado para download. Note que a elaboração deste documento deve ser feita em inglês. Acreditamos que este procedimento desenvolva também habilidades de escrita para estudantes. Contamos com sua colabaração!
